Core Values

BUDAYA KERJA KSP WADIRA

Untuk mencapai visi dan menjalankan misi bisni KSP Wadira, kami berkomitmen untuk memberikan pelayanan yang mudah, cepat dan terpercaya dengan berpegang pada nilai-nilai perusahaan yang kami sebut sebagai Budaya Kerja WADIRA, yaitu :

Wiweka : Bekerja dengan bijaksana penuh kehati-hatian (prudent). Aman : Bekerja dengan tanggung jawab dan penuh ketelitian. Dedikasi : Bekerja dengan rasa bangga penuh ketekunan. Integritas : Bekerja dengan segenap hati penuh kejujuran. Ramah : Bekerja dengan senyum ikhlas penuh ketulusan. Antusias : Bekerja dengan proaktif-kreatif penuh kesungguh-sungguhan.
